Minerales en Minecraft

Diamante

El mineral más valioso para herramientas y armaduras avanzadas.

Oro

Se utiliza para herramientas, relojes y decoración.

Hierro

Muy útil para herramientas y armaduras de nivel intermedio.

Redstone

Ideal para circuitos y mecanismos.

Carbón

Fundamental para crear antorchas y como combustible.
